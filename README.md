# Tic Tac Toe — with Ishy 🎮

A clean, self-contained Tic Tac Toe game. **No build step, no dependencies** — just one HTML file.

## How to play

Double-click **`index.html`** (or open it in any browser). That's it.

## Features

- **2-Player mode** — pass & play on the same device (X vs O).
- **Vs Computer mode** — play against the machine:
  - **Easy** — the computer plays random moves (winnable, good for kids).
  - **Unbeatable** — the computer uses the **minimax** algorithm and never loses.
- **Choose your symbol** — play as X (go first) or O (go second) against the computer.
- **Scoreboard** — tracks wins for X, O, and draws across rounds.
- **Winning line highlight**, smooth animations, and a mobile-friendly responsive layout.

## Tech

- A single `index.html` — HTML, CSS, and vanilla JavaScript inline.
- Zero dependencies. Works offline.
